# Full Stack Personal Portfolio

A premium, professional portfolio website built with React, Tailwind CSS, and Framer Motion. Featuring an immersive animated background, project showcase with live screenshots, and a functional contact form.

## 🚀 Features
- **Modern UI/UX**: Professional glassmorphism design with an Emerald Green theme.
- **Animated Background**: Mesmerizing floating gradient blobs and noise texture.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Dynamic Projects**: Showcases live projects with tags and links.
- **Working Contact Form**: Integrated with Formspree for direct email delivery.

## 🛠️ Tech Stack
- **Frontend**: React, Vite, Framer Motion, Tailwind CSS, Lucide Icons.
- **Backend**: Node.js, Express (ready for expansion).
- **Deployment**: Vercel.

## 📦 Project Structure
- `/client`: React frontend source code and assets.
- `/server`: Node.js backend source code.

## 🔧 Installation & Setup
1. Clone the repository.
2. Install dependencies for both client and server:
   ```bash
   cd client && npm install
   cd ../server && npm install
   ```
3. Run the development server:
   ```bash
   cd client && npm run dev
   ```
